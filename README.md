# CashCompass

🌍 The Ultimate Bitcoin Cash Resource Hub

## Live Website

https://cashcompass-bch.vercel.app

## About

CashCompass is a community-driven resource hub for the Bitcoin Cash ecosystem.

Its mission is to make Bitcoin Cash easier to discover by bringing together wallets, merchants, podcasts, news sources, educational resources, community leaders, development tools, and BCH projects from around the world.

## Features

- BCH Global Network
- Community Ambassadors
- BCH Resources Directory
- Wallet Resources
- Podcasts & Media
- Development Tools
- Educational Resources
- BCH Projects

## Why CashCompass?

The Bitcoin Cash ecosystem is growing globally, but many valuable resources are spread across different platforms.

CashCompass helps users find everything they need in one place.
